# VICE

VICE is a KMP MVI framework that is built on Compose.

### Setup

```
repositories {
  mavenCentral()
}

dependencies {
  implementation("com.eygraber:vice-core:0.0.2")
}
```

Snapshots can be found [here](https://s01.oss.sonatype.org/#nexus-search;gav~com.eygraber~vice~~~).
